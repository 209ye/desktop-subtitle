这是一个基于阿里云实时语音转写实现的桌面字幕。

代码实现非常简单，根据阿里云开发文档里面的demo代码复制粘贴就完成了。使用 `JavaFx` 实现透明窗口 + Label。

实时语音识别的效果不错，可以用于录制教学视频。

源代码：[https://github.com/yi-ge/desktop-subtitle ](https://github.com/yi-ge/desktop-subtitle)
